# NetBeans

## JDK setzen

In `/Applications/NetBeans/NetBeans\ 8.2.app/Contents/Resources/NetBeans/etc/netbeans.conf`:
```
netbeans_jdkhome="/Library/Java/JavaVirtualMachines/jdk1.8.0_144.jdk/Contents/Home"
```
